# Trade-Bills-and-Bipartisanship-
CSV file for dataset of for trade bills, roll call votes, and bipartisanship score. 
## Dataset Description

## Dataset Variables

| Column Name | Description |
|------------|------------|
| Legislation Name | Official name or title of the bill. |
| Congress | Congressional session in which the bill was considered. |
| Month-Day H | Month and day the bill passed the House of Representatives. |
| Month-Day S | Month and day the bill passed the Senate. |
| Year | Calendar year of the vote or passage. |
| Bill | Official bill number (e.g., H.R. or S. designation). |
| Vote #H | Roll-call vote number in the House, sourced from Voteview. |
| Vote #S | Roll-call vote number in the Senate, sourced from Voteview. |
| Description/Notes | Additional context or relevant details about the legislation. |
| Supports Free Trade |  TRUE if a “yes” vote supports free trade; FALSE if a “yes” vote opposes free trade. |
| House | Summary of the overall vote breakdown in the House. |
| Passed or Failed | Indicates whether the bill ultimately passed or failed. |
| Dem Yeas-Nays | Vote breakdown among Democratic members (yeas vs. nays). |
| Rep Yeas-Nays | Vote breakdown among Republican members (yeas vs. nays). |
| Minority Party H | Political party holding minority status in the House at the time of the vote. |
| Bipartisanship Score (H) | House bipartisanship score calculated using Tama (2023); ranges from 0 (fully partisan) to 1 (fully bipartisan). |
| % of Dem Vote Free Trade | Percentage of Democratic members voting in favor of free trade. |
| % of Rep Vote Free Trade | Percentage of Republican members voting in favor of free trade. |
| Senate | Summary of the overall vote breakdown in the Senate. |
| Minority Party S | Political party holding minority status in the Senate at the time of the vote. |
| Bipartisanship Score (S) | Senate bipartisanship score calculated using Tama (2023); ranges from 0 (fully partisan) to 1 (fully bipartisan). |
| Source | Original source of the legislative or voting data. |
| Voteview Link | Direct link to the corresponding Voteview record for the bill. |
